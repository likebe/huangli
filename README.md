# 玄历 · 交互式黄历 + 五行八字

一个**单文件、零依赖、可离线**的网页应用：可交互日历 + 详细黄历（宜忌/冲煞/值神/建除/二十八宿/彭祖百忌/胎神/时辰吉凶）+ 五行八字排盘 + 所有专有名词点击可看通俗解释 + 择吉问答（内置规则引擎，并可选接入 AI）。

> 打开就是一个 `index.html`，双击即可用；上传到 GitHub Pages 即可得到一条可分享的网址。

---

## ✨ 功能一览

- **可交互日历**：公历/农历对照，逐月逐年翻看（1900–2100），点任意一天看详情；今日、节气、农历节日、黄道/黑道一眼可辨。
- **详细黄历**：日柱干支与纳音、岁次、生肖、值神（黄道吉/黑道凶）、建除十二神、二十八宿（含七曜）、月相、宜忌、冲煞与煞方、正冲干支、彭祖百忌、胎神方位、十二时辰吉凶。
- **五行八字排盘**：输入出生日期、时辰与性别，得到年/月/日/时四柱、**十神、地支藏干、纳音、五行分布柱状图、日主旺衰**，以及**大运排盘（顺逆排＋起运＋每步十神）**；可一键保存生辰到本机。
- **吉神方位**：每日财神／喜神／福神／阳贵神／阴贵神方位，办事可朝吉方。
- **择日助手**：选事项（嫁娶／出行／动土／开市／搬家／祈福／安葬／求医…）与日期范围，自动按「宜忌＋值神＋建除＋黄道」批量列出吉日，点选即跳转查看当天详情。
- **名词通俗解释**：凡带虚线的词都可点击，弹出"一句话讲明白"的释义（共 50+ 词条）。
- **择吉问答**：选日期 + 选领域（嫁娶/出行/动土/开市/搬家/安葬/祈福/求医）或直接打字提问（可带"我属X"），给出结论、理由与推荐吉时；并可选接入你自己的 AI 接口做自由问答。
- **四大分页**：黄历日历 ｜ 八字排盘 ｜ 择日助手 ｜ 黄历问答，界面清爽。
- **可分享链接**：地址带 `#d=2026-06-15`，别人打开即跳到同一天。

## 🚀 用法一：本地直接用

直接双击 `index.html`，用任意现代浏览器（Chrome / Edge / Safari / Firefox）打开即可。无需联网、无需安装。

## 🌐 用法二：发布到 GitHub Pages，得到可分享网址（推荐）

整个过程约 5 分钟，**全程网页操作，不用装任何软件**。

### 第 1 步：注册 / 登录 GitHub
打开 https://github.com ，注册并登录一个账号。

### 第 2 步：新建仓库（Repository）
1. 右上角 **「＋」→ New repository**。
2. **Repository name** 填一个名字，例如 `huangli`。
3. 选择 **Public（公开）**（GitHub Pages 免费版需要公开）。
4. 点 **Create repository**。

### 第 3 步：上传 index.html
1. 在新仓库页面点 **「Add file」→「Upload files」**。
2. 把本目录里的 `index.html` 拖进去（**文件名必须是 `index.html`**）。
3. 下方点 **Commit changes**。

### 第 4 步：开启 Pages
1. 进入仓库的 **Settings（设置）**。
2. 左侧菜单点 **Pages**。
3. **Build and deployment → Source** 选 **Deploy from a branch**。
4. **Branch** 选 **`main`**，文件夹选 **`/ (root)`**，点 **Save**。

### 第 5 步：拿到你的网址
等 1–3 分钟刷新 Pages 页面，顶部会出现：

```
https://你的用户名.github.io/huangli/
```

这就是可以发给任何人的链接。想直接分享某一天，在后面加上日期即可：

```
https://你的用户名.github.io/huangli/#d=2026-06-15
```

> 之后要更新内容，只需在仓库里重新上传同名 `index.html`，几分钟后网址自动更新。

## 🤖 可选：接入 AI 做自由问答

应用默认使用**内置规则引擎**（离线、免费、不需要任何密钥），已能回答"某天是否适合做某事"。
如果想要更自然的自由对话，可在问答区底部「⚙ 进阶」里：

1. 模式选 **AI 自由问答**；
2. 填入**接口地址**（任意 OpenAI 兼容接口，如 `https://api.openai.com/v1/chat/completions`）、**模型名**（如 `gpt-4o-mini`）、**你的密钥**；
3. 点 **保存设置**。

> 密钥只保存在你本机浏览器（localStorage），不会上传到任何服务器。⚠ 注意：纯静态网页里调用 AI 时密钥会出现在浏览器请求中，请勿在公开电脑上填写、勿把密钥写进公开仓库。

## ✅ 数据准确性说明

本工具的**历法类数据是用公开天文/历法算法精确推算**，并已逐日与权威天文历表（寿星天文历 sxtwl）核对：

- 农历、年/月/日干支、二十四节气：在 **1950–2099 共 3289 天** 的逐日比对中 **100% 一致**；二十四节气在 **1900–2100** 内采用历表精确日期。
- 建除、值神（黄黑道）、二十八宿、纳音、冲煞、正冲、胎神、彭祖百忌：与新浪、bmcx 等老黄历多日交叉核对一致（例：2026-06-15 庚申日、满日、青龙黄道、毕月乌、冲虎(甲寅)煞南、胎神碓磨炉外东南，完全吻合）。
- 二十八宿值日另有内在校验：每宿的"七曜"恒与星期对应（"月"宿必周一、"水"宿必周三……），本工具全部满足。
- 八字与方位另用 **lunar-javascript（6tail）** 独立复核：8 组不同性别/年份的命例，四柱、十神、大运、起运、财神/喜神/福神/贵神方位全部一致。

**关于"宜忌"**：传统通书的宜忌由数十种神煞叠加而成、各流派略有差异。本工具的宜忌**以"建除十二神"的传统对应推演**，清晰、可解释、可复现，但可能与某本具体通书不完全相同，仅供参考。命理/择吉内容属传统文化范畴，请理性看待，重大决策请综合判断。
